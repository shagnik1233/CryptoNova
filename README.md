# 🚀 CryptoNova | Live Cryptocurrency Tracker

A modern, fast, and responsive Single-Page Application (SPA) built with Vanilla JavaScript, HTML, and CSS that tracks cryptocurrency markets in real-time.

- **Real-Time Data**: Fetches live prices, market caps, and 24h volumes directly from the CoinGecko API.
- **Single-Page Application (SPA)**: Smooth, instant navigation between sections without any full page reloads.
- **Dark Mode UI**: A sleek, premium dark theme utilizing Glassmorphism (frosted glass) effects and smooth micro-animations.
- **Top Markets**: View the top 100 cryptocurrencies sorted by market cap.
- **Trending Coins**: Discover which coins are the top searched in the last 24 hours.
- **Live Search**: Instantly filter and find specific cryptocurrencies on the market table.
- **Global Metrics**: Monitor the total global market cap, 24h trading volume, and Bitcoin dominance at a glance.
- **Fully Responsive**: Optimized flawlessly for mobile, tablet, and desktop viewing.

## 🛠️ Built With

- **HTML5** - Semantic page structure.
- **CSS3** - Custom styling, CSS Variables, Flexbox/Grid, and Keyframe Animations.
- **Vanilla JavaScript (ES6+)** - Logic, Async/Await API fetching, and custom DOM routing.
- **CoinGecko API** - Free, reliable cryptocurrency data.
- **Google Fonts** - 'Outfit' and 'Inter' typefaces.
- **FontAwesome** - Vector icons.
